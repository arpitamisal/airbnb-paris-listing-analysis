# Airbnb Paris Listing Analysis (Python)

This project analyzes Airbnb listings in Paris to explore pricing trends, neighborhood differences, host growth patterns, and the impact of regulations over time using data aggregation and visualization.

## Tech Stack
- Python  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Jupyter Notebook  

## Key Work
- Data cleaning, profiling, and datatype corrections  
- Filtered dataset to include Paris listings only  
- Aggregated listings by neighbourhood to compute average price  
- Analyzed price trends by accommodation capacity  
- Examined host growth and average price trends over time  
- Built visualizations to identify market patterns and regulatory impact  

The notebook demonstrates end-to-end data analysis, from raw dataset preparation to structured aggregation and insightful visual storytelling.
